Afbeeldingen toevoegen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de develop branch zit.

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Vervang bovenaan de pagina "Naam van student" door jouw eigen naam (id=naam)
  • Commit deze wijziging
  • Zoek een passende afbeelding voor DIT hoofdstuk, voeg deze afbeelding toe in de map images en maak een commit.
  • Pas de src aan van de placeholder afbeelding van DIT hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is en maak een commit.
  • Ga terug naar de develop branch.
  • Merge de aanpassingen van afbeeldigen naar develop.
  • Push de develop branch naar github

Use case diagram

  • Start een nieuwe branch use-case-diagram vanuit develop
  • Maak een Use Case Diagram voor de onderstaande User-stories
    Maak hierbij gebruik van:
    • Include relaties
    • Extend relaties
    • Primaire & secundaire actor(s)
  • Vervang onderstaande placholder image (id=use-case-image) met jouw use case diagram.
  • Commit deze aanpassing
  • Merge de aanpassingen van use-case-diagram naar de develop branch
  • Push de develop branch naar github

User stories

  • Als student kan ik inloggen in de app en dan mijn school kiezen.
  • Als student kan ik mijn agenda van vandaag zien.
  • Als student kan ik door mijn agenda navigeren, vooruit en achteruit, zodat ik snel een andere afspraak kan inzien.
  • Als student kan ik een overzicht zien met per vak mijn laatst behaalde resultaat en het eindresultaat.
  • Als student kan ik in het overzicht zien welke resultaten nieuw zijn.
  • Als student kan ik detailinformatie zien van elk behaalde resultaat.
  • Als student zie ik een overzicht van berichten die aan mij, of aan de groepen waarvan ik deel uitmaak, gestuurd zijn.
  • Als student kan ik de volledige inhoud van een bericht bekijken.
  • Als student kan ik een bericht verwijderen.
  • Als student kan ik zien welke berichten nieuw zijn vergeleken met de vorige keer.
  • Als student kan ik mijn personalia inzien.
  • Als student kan ik mijn personalia wijzigen zoals: telefoonnummer, emailadres en pasfoto.
  • Als student kan ik in mijn agenda zien bij welke les ik huiswerk / een toets heb.
  • Als student kan ik detailinformatie van een huiswerkopdracht zien.
  • Als student kan ik detailinformatie van een toets zien.
  • Als student kan ik een absentiemelding doen.
  • Als mentor kan ik absentiemeldingen van studenten ontvangen
